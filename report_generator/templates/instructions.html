Instructions

  1. First, download your raw data from Thoughtexchange by going under 'Reports' in your Thoughtexchange dashboard and clicking the 'Generate' button under the section of the page titled 'Data Download.'
  2. After completing the download, you will receive a ZIP file contaning 3 CSV files, or spreadsheets. Unzip this file and locate the file titled 'exchangedata.csv' within the resulting folder. This is the file that you will upload to the Thoughtexchange Feedback Report generator.
  3. Before uploading, make sure that the 'exchangedata.csv' file has not been edited and that the first row is formatted as follows: 'Exchange question', 'Thought (original)', 'Star score - overall', 'Rank - overall', 'Moderation status', 'Moderation Reported by'. Note that each phrase separated by a comma represents a column in the CSV file.
  4. You are now ready to use the Thoughtexchange Feedback Report generator! Just upload your file and the report will be generated.
